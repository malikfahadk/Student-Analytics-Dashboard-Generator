# Student Analytics Dashboard Generator

An interactive academic decision-support dashboard built with React, TypeScript, Vite, data visualization, and machine-learning analytics.

## Deployment

This repository is configured for a free Hugging Face Static Space. Hugging Face installs the Node.js dependencies, runs `npm run build`, and serves the generated `dist/index.html`.
